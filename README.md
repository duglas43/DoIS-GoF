## Описание

Проект предназначен для выполнения домашнего задания по проектированию инфокоммуникационных систем с использованием шаблонов проектирования GoF (Gang of Four).

## Как запустить проект

1. Docker Compose

```bash
docker compose up
```

2. Node.js

```bash
npm install
npm run start:dev
```
